Git

  • git init - Создание нового репозитория Git в пустой директории или превращение существующего директория в новый репозиторий Git
  • git clone - клонирование репозитория из удалённого хранилища на локальную машину
  • git push - выгрузка изменений в удалённый репозиторий
  • git pull - получение изменений из удалённого репозитория на локальную машину
  • git status - просмотр списка изменнных файлов
  • git branch - создание ветки
  • git checkout - переход в ветку
  • git commit - сохранение изменений в Git
  • git log - просмотр историй коммитов в репозитории
